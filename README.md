# DAS-TOOLS Website

A web site for batch query registration of [DAS](https://da.systems) domain names
Please visit: [https://www.das.tools](https://www.das.tools)

## Quick Start

### install dependencies

```sh
npm install
# or
yarn add
```

### Run

```sh
npm run dev
```

### Build

```sh
npm run build
```
